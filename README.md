# 💱 Currency Converter

A simple, fast currency converter app with live exchange rates, built with Streamlit.

## ✨ Features

- 📊 Live exchange rates from Frankfurter API
- 🔄 Auto-refresh every 5 minutes
- 📋 Complete rates table for all currencies
- 🎨 Clean, modern UI with emojis
- ⚡ Fast and lightweight

## 💰 Supported Currencies

| Code | Currency Name |
|------|---------------|
| USD | US Dollar |
| EUR | Euro |
| GBP | British Pound |
| JPY | Japanese Yen |
| TRY | Turkish Lira |
| CNY | Chinese Yuan |
| TMN | Iranian Toman |

## 🚀 Quick Start

### Prerequisites

- Python 3.8 or higher
- pip package manager

### Run the app
streamlit run app.py
